페이지 1번(메인페이지)

1.(헤더)
	기능
	(기본페이지)
	1. 추천 랭킹에서  추천 버튼 누르면 1페이지 이동 랭킹 버튼 누르면 2페이지 이동 이때 글자 굵기 커짐 및 언더바 생김
	2.shop 버튼 누르면 2페이지 이동 글자 굵기 커짐 home 버튼 누르면 1페이지 이동 글자 굵기 커짐	
	3.돋보기 클릭시 검색창 나옴 왼쪽 위의 x 버튼 클릭시 나와짐
	(가로 사이즈가 765px 일때)
	1.밑에 아래창에 새로운 버튼들이 생김 / 샵버튼 누르면 2페이지로 이동 및 색 변경	
2.(메인베너)
	기능
	(기본페이지)	
	1.그림이 2초마다 바뀜 마우스가 사진에 들어오면 바뀌는거 멈춤
	2.밑에 인디케이터 클릭시 해당 그림으로 이동
	3.화살표 클릭시 사진 바뀜 
3.(메인 컨텐츠)
	기능
	(기본페이지)
	1. just dropped 부분에 더보기 누르면 한번 아이템이 나오고 더보기 사라짐 
	2. 아이템 클릭시 3페이지 아이템 창으로 이동
	3. most popular  부분에 더보기 누르기 두번 하면 더보기 사라짐
	4. 코디 부분에 사진에 마우스 가져가면 화살표 생기고 누르면 사진이 움직이면서 바뀜
	4.(푸터)
	1. 각 페이스북 인스타 카카오톡 클릭시 사이트 이동
	(가로 사이즈 960px 일때)
	2.  눈에 보이던 사업자 정보가 없어지고 클릭시 나오는 사업자 정보 란이 생김
페이지 2번(샵 페이지)

1.(헤더와 푸터는 동일)

2.(메인 페이지) 
	기능
	1.필터 부분의 사이드 란을 헤더 아래쪽이랑 닿으면 따라다님 그러다가 푸터 위쪽에 오면 고정됨 
	2. 필터 부분의 + 버튼 누르면 밑에 정보 보이고 안보이고 기능 
	3.상품란 16개의 상품이 3번 무한 스크롤 됨

	(가로 사이즈 960px 일때)
